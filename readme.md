Conventional Commits
init: 初始化
feat: 新增功能
fix: 修復 bug
style: 調整樣式（不影響邏輯）
refactor: 重構程式碼
docs: 更新文件或 README
chore: 雜事（設定檔、套件更新）
test: 新增測試

feat: 新增練習檔案
fix: 改掉寫錯的地方
docs: 更新筆記或 README

格式規則：

全部小寫
冒號後面一個空格
說明簡短，不要超過 50 個字
不用句號結尾

ex:
# 修改 about 頁面文字大小
style: adjust about page font-size

# 新增聯絡我們頁面
feat: add contact page

# 修好導覽列跑版
fix: fix navbar layout issue

# 新增陣列練習檔案
feat: add array methods practice
